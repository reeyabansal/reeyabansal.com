Predicting Electricity Consumption using Weather Data (2024)

Technical Skills: R, Linear Regression, Statistical Analysis, ANOVA

Motivation & Impact: This project aimed to analyze the relationship between weather conditions and household electricity consumption in Sceaux, France. I developed multiple linear regression models using R to process a large-scale dataset containing over 2 million measurements of power consumption and weather variables. Through statistical analysis, I identified temperature, humidity, and daylight as key predictors of energy usage. I validated the models with ANOVA tests and residual analysis to ensure their reliability. The findings provide valuable insights for optimizing energy management, helping reduce costs in response to weather variations.

This project showcases my proficiency in statistical analysis, data modeling, and R programming for practical applications in energy consumption prediction.

Disease Diagnostic Software (2023)

Technical Skills: Python (tkinter), Graph Data Structures, Tree Algorithms

Motivation & Impact: This project developed an interactive diagnostic tool to predict potential diseases based on user-reported symptoms using Python. I implemented a custom tree data structure, DiseaseTree, to model relationships between symptoms and diseases. The system also included a graph-based structure, PrecautionGraph, to analyze health data and generate personalized health precautions. With an intuitive tkinter-based user interface, users could input symptoms and visualize the disease likelihoods and recommended precautions. I designed algorithms to calculate disease probabilities based on symptom patterns and traverse complex symptom-disease relationships efficiently.

This project demonstrates my strong algorithmic thinking, data structure implementation, and the practical application of computer science concepts to healthcare challenges.
